# Java Calculator

Javaで作成した、簡単な計算機アプリです。

## 概要
コンソール上で2つの数字を入力し、四則演算（`+`, `-`, `*`, `/`）を選ぶことで計算結果を表示します。

## できること
- 2つの数字の入力
- 四則演算（`+`, `-`, `*`, `/`）
- 数字以外が入力された場合のエラー処理
- 0で割ろうとした場合のエラー処理
- 続けて計算するかどうかの選択

## 使用技術
- Java
- Scanner

## 実行方法

```bash
javac Main.java
java Main
```
## 実行例
```text
1つ目の数字を入力してください：10
2つ目の数字を入力してください：5
演算子を入力してください (+, -, *, /) --> +
結果：15
続けますか？ (y/n) --> n
```

## 工夫した点
- `Scanner` を使って入力を受け取れるようにした
- `while` を使って、無効な入力があった場合に再入力できるようにした
- `switch` を使って、演算子ごとに処理を分けた
- 0で割る場合にエラーメッセージを表示するようにした
