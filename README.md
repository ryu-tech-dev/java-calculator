# Java Calculator

Javaで作成した、簡単な計算機アプリです。
コンソール版の作成から始め、メソッド化による整理、さらにSwingを使ったGUI版の作成まで行いました。

## 概要
このアプリは、2つの数字を入力し、四則演算（`+`, `-`, `*`, `/`）を行う計算機アプリです。
最初はコンソール上で動作する形で作成し、その後、入力処理や計算処理をメソッド化して整理しました。
さらに、Swingを用いてGUI版にも発展させました。

## できること
### コンソール版
- 2つの数字の入力
- 四則演算（`+`, `-`, `*`, `/`）
- 数字以外が入力された場合のエラー処理
- 0で割ろうとした場合のエラー処理
- 続けて計算するかどうかの選択
- 入力処理、計算処理、継続確認のメソッド化

### GUI版
- 2つの数字を入力欄に入力
- `+`, `-`, `*`, `/` のボタンで計算
- 計算結果を画面に表示
- 数字以外が入力された場合のエラー表示
- 0で割ろうとした場合のエラー表示

## 使用技術
- Java
- Scanner
- Swing

## 実行方法
```bash
javac Main.java
java Main
```

## 実行例

### コンソール版
```text
1つ目の数字を入力してください：10
2つ目の数字を入力してください：5
演算子を入力してください (+, -, *, /) --> +
結果：15
続けますか？ (y/n) --> n
```

### GUI版
- 1つ目の数字と2つ目の数字を入力
- 演算子ボタン (`+`, `-`, `*`, `/`) を押す
- 結果が画面下部に表示される
  
## 工夫した点
- `Scanner` を使って入力を受け取れるようにした
- `while` を使って、無効な入力があった場合に再入力できるようにした
- `switch` を使って、演算子ごとに処理を分けた
- 0で割る場合にエラーメッセージを表示するようにした
- メソッド化によって、`main` の中を読みやすく整理した
- Swingを使ってGUI版を作成し、ボタン操作で計算できるようにした

## 学んだこと
- メソッドに分けることで、処理の役割が整理されて読みやすくなること
- GUIでは `Scanner` ではなく、`JTextField` などの画面部品から入力を受け取ること
- `add` で部品を追加し、`setLayout` で配置を決めること
